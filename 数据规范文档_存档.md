**数据规范文档（Data Specification，数据规范）**

规定程序命名及数值类型

**ST_Character**

  --------------------------------
  **字段**   **类型**   **说明**
  ---------- ---------- ----------
  HP         Float      血量

  Location   Vector     位置

  Time       Float      时间
  --------------------------------

**ST_WorldState**

  --------------------------------
  **字段**   **类型**   **说明**
  ---------- ---------- ----------
  GUID       String     唯一ID

  State      Bool       状态
  --------------------------------

**DT_Chapter**

  ------------------------------------
  **字段**      **类型**    **说明**
  ------------- ----------- ----------
  ChapterName   String      章节名

  Image         Texture2D   预览图
  ------------------------------------

**台词Data**

  -----------------------------------
  **字段名**   **类型**    **说明**
  ------------ ----------- ----------
  ID           Int32       台词ID

  Speaker      String      说话角色

  LineText     Text        台词内容

  VoiceAsset   SoundWave   语音资源

  Chapter      E_Chapter   所属章节
  -----------------------------------

Time Float 台词播放时间

规范文档会规定：

命名规则\
字段规则\
数据类型\
默认值\
取值范围

例如：

\

\
